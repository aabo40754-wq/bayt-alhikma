# بيت الحكمة للنشر والتوزيع

موقع تعريفي لدار نشر وتوزيع، مبني بـ React و Tailwind CSS، بتصميم عربي (RTL).

## الأقسام

- الرئيسية (Hero)
- إحصائيات الدار
- من نحن
- خدماتنا (النشر، التوزيع، التحرير، حقوق الترجمة)
- إصداراتنا (نخبة من الكتب)
- تواصل معنا (نموذج تواصل — واجهة فقط، بلا خادم)

## التشغيل محليًا

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```

ينتج ملفات الإنتاج في مجلد `dist/`.
